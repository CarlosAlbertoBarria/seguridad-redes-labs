# Seguridad Redes Labs

Repositorio personal de laboratorios y documentación práctica sobre diseño de redes seguras, Linux y fundamentos de seguridad informática.

## Objetivo

Construir una base práctica y verificable en redes, seguridad aplicada, análisis de topologías, identificación de riesgos y diseño de medidas de protección.

Este repositorio forma parte de mi proceso de formación en Seguridad Informática, con foco en documentación técnica, prácticas de laboratorio y aprendizaje progresivo.

## Metodología

- Documentación escrita en Markdown.
- Diagramas y simulaciones con Cisco Packet Tracer.
- Evidencias mediante capturas de pantalla.
- Uso de Git para versionar avances.
- Un commit por avance relevante.
- Explicación de cada decisión técnica.
- Enfoque progresivo: fundamentos, diseño, riesgos, controles y hardening.

## Contenido

### Módulo 1 - Diseño de redes seguras

- Unidad 1: Introducción a redes informáticas.
- Unidad 2: Instalación y configuración en redes.
- Unidad 3: Potenciales riesgos.
- Unidad 4: Seguridad aplicada.
