# Módulo 1 - Diseño de redes seguras

Este módulo reúne prácticas y documentación sobre fundamentos de redes, configuración, riesgos y medidas de seguridad aplicadas al diseño de infraestructuras.

## Unidades

1. Introducción a redes informáticas.
2. Instalación y configuración en redes.
3. Potenciales riesgos.
4. Seguridad aplicada.

## Objetivo del módulo

Comprender cómo se diseñan, analizan y protegen redes informáticas, identificando dispositivos, servicios, puntos críticos, amenazas y controles de seguridad.
