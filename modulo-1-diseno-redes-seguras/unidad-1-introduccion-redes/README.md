# Unidad 1 - Introducción a redes informáticas

## Objetivo

Comprender los conceptos fundamentales de redes informáticas, tipos de redes, topologías, dispositivos principales y puntos de incidencia que pueden afectar la disponibilidad, integridad o confidencialidad de la información.

## Temas principales

- Concepto de red informática.
- Historia y evolución de las redes.
- Tipos de redes.
- Topologías.
- Modelo OSI.
- Modelo TCP/IP.
- Dispositivos de red.
- Puntos de incidencia.
- Relación entre redes y seguridad informática.
